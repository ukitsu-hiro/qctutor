量子コンピュータ TUTOR

ゲート型量子コンピュータの基礎的な動きを学ぶためのツールです。

特徴は以下の通り：

・あらかじめ Toffoli や Glover、フーリエ変換、足し算回路などの回路定義が用意されている

・GUIでゲートを入れ替えたり、増やしたり、減らしたり、位置を変えたり、コピー＆ペーストしての編集が可能

・マトリックス計算、状態計算、ブロッホ球表示、ブロッホ球上の遷移を表示可能

注意：

・簡単な動作チェックを行っていますが、まだ十分ではありません。

・計算方法が誤っている可能性があります。

使い方：

・qctutor-*.zip を展開してください。

・Windows 版は python3.13 が必要です。do_main.bat を実行してください。
  VirusTotalで検査済み。2/66 flags (Google, Ikarus のみ) は問題ありません。

・Mac 版は python3.13 が必要です。do.command を実行してください。
  セキュリティチェック確認が何度も出るので、Windowsで展開してからMacへコピーするのがおすすめです。
  VirusTotalで検査済み。
